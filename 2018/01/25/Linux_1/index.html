Linux常用命令(一)：目录基本操作

发表于 2018-01-25

今天跟着老大去生产环境上了一下补丁，发现好多Linux命令不熟，整理一下方便查阅。

Terminal简单操作

  • 上/下方向 键：浏览历史执行命令
  • Home/End 键：移动光标到本行开头/结尾
  • Ctrl + c ：终止当前程序
  • Tab 键：输入部分命令或文件名，按Tab可以自动补全
  • * 键 ：模糊匹配，当要操作的文件或目录很长时，可以使用*号匹配
  • man 查看命令详情： man tar

目录基本操作

权限

  • 访问者(UGO模型)
    • 文件和文件目录的所有者:UUser
    • 文件和文件目录的所有者所在的组的用户:GGroup
    • 其它用户:OOthers
  • 事物属性
权限 缩写 全写 数字表示 对象是文件 对象是目录
读 r Read 4 可以读取文件内容 可以浏览该目录信息
写 w Write 2 可以修改文件内容 可以删除移动目录内文件
执行 x Execute 1 可以执行文件 可以进入目录

ls

  • 原意：list
  • 作用：显示目录文件
  • 常用：

cd

  • 原意：change directory
  • 作用：切换目录
  • 常用：

pwd

  • 原意：print working directory
  • 作用：显示当前目录
  • 常用：

mkdir

  • 原意：make directories
  • 作用：创建新目录
  • 常用：

rmdir

  • 原意：remove empty directories
  • 作用：删除空目录
  • 常用：

rm

  • 原意：remove
  • 作用：删除一个目录/文件。无法恢复rm删除的文件。rm命令可以用-i选项，输入y并按Enter键，才能删除文件。指定被删除的文件列表，如果参数中含有目录，则必须加上-r或者-R选项。
  • 常用：

cp

  • 原意：copy
  • 作用：复制文件或目录
  • 常用：

mv

  • 原意：move
  • 作用：剪切文件/重命名
